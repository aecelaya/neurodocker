neurodocker.cli.generate module¶

class neurodocker.cli.generate.GroupAddCommonParamsAndRegisteredTemplates(*args, **kwds)[source]¶

Bases: click.core.Group

Subclass of click.Group that adds parameters common to reproenv generate commands, registers templates, and adds parameters to render templates.

Note: Commands that are part of this group must be called using the group. This is because the .get_command() method of this group adds required parameters.
