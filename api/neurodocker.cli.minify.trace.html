neurodocker.cli.minify.trace module¶

Minify a Docker container.

This program removes all files under specified directories in the container _not_ used by the given command(s).

neurodocker.cli.minify.trace.copy_file_to_container(container: Union[str, docker.models.containers.Container], src: Union[str, pathlib.Path], dest: Union[str, pathlib.Path]) → bool[source]¶

Copy local_filepath into container:container_path.

Parameters

  • container (str or docker.models.containers.Container instance) – Container to which file is copied.

  • src (str or pathlib.Path instance) – Filepath on the host.

  • dest (str or pathlib.Path instance) – Directory inside container. Original filename is preserved.

Returns

success – True if copy was a success. False otherwise.

Return type

bool
